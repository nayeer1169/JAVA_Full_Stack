Tech Glossary

HTML
HTML stands for HyperText Markup Language, and it is the standard code used to structure and display content on the web. Rather than acting as a traditional programming language with computational logic, it acts as a "skeleton" or structural foundation. It uses a system of text tags to tell your web browser exactly how to display headers, paragraphs, links, images, and other multimedia.
CSS
CSS (Cascading Style Sheets) is a stylesheet language used to control the visual presentation, design, and layout of a document written in a markup language like HTML. Alongside HTML and JavaScript, it is a core technology of the World Wide Web. While HTML builds the underlying framework and structural content of a page, CSS applies rules to manipulate visual attributes like fonts, colors, spacing, positioning, and layouts
